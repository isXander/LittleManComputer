PC — Program Counter

Holds the address of the next instruction the computer will fetch.

After every fetch, it automatically ticks up by 1, so the machine naturally walks through memory in order.

Jump instructions (BRA, BRZ, BRP) overwrite it to make the program skip to a different line.

MAR — Memory Address Register

A sticky note with the mailbox number the CPU is about to read from or write to.

Think of it as: "which mailbox am I pointing at right now?"

MDR — Memory Data Register

A temporary buffer that carries a value between memory and the rest of the CPU.

On a read, the mailbox copies its content into the MDR. On a write, the CPU puts a value here and memory picks it up.

IR — Instruction Register

Stores the 3-digit instruction that was just fetched, so the CPU can look at it and decide what to do.

The first digit tells the CPU what kind of instruction it is (add, store, branch…) and the last two digits are usually an address.

ACC — Accumulator

The computer's single "scratch pad" for numbers. Almost every arithmetic operation uses it.

LDA loads a value into it, ADD/SUB change it, STA saves it to memory, and OUT prints it.

Values range from -999 to 999.

📖 LMC Instruction Cheat Sheet

INP901Read a number from the input and put it in the accumulator.
OUT902Print the value currently in the accumulator.
OTC922Print the accumulator as a character (e.g. 72 → "H"). Non-standard extension.
LDA xx5xxLoad — copy the contents of mailbox xx into the accumulator.
STA xx3xxStore — copy the accumulator into mailbox xx (also written STO).
ADD xx1xxAdd the value in mailbox xx to the accumulator.
SUB xx2xxSubtract the value in mailbox xx from the accumulator.
BRA xx6xxBranch always — jump: next instruction comes from address xx.
BRZ xx7xxBranch if zero — jump to xx only if the accumulator is exactly 0.
BRP xx8xxBranch if positive — jump to xx if the accumulator is 0 or more.
HLT000Stop the computer. The Little Man goes home.
DAT n—Not an instruction: reserves a mailbox to hold a value n (default 0). Used for variables.
A label at the start of a line gives that mailbox a name you can reference. Example: COUNT DAT 0  →  then later LDA COUNT.
